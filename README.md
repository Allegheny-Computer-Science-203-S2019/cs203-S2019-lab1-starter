# cs203-S2019-lab1-starter

This repository contains the assignment sheet for the first laboratory
assignment in [Computer Science 203 Spring
2019](https://www.gregorykapfhammer.com/teaching/cs203S2019/). Please use the
`report.md` file to document the ways in which you contributed to the creation
of the assessment sheet and the conduct guide that we will use during all of the
software projects in this course. You will receive feedback on your work and be
graded on a checkmark basis for this laboratory assignment. Whenever possible,
you should describe the your efforts on this laboratory assignment according to
the following levels:

* N = None
* I = Inadequate
* A = Adequate
* G = Good
* E = Excellent
